# Kora Launcher – Privacy Policy

**Effective Date:** March 24, 2026

Kora Launcher is designed to help users reduce screen time and use their devices more intentionally. Your privacy is a core principle of this app.

## 1. Data Collection and Usage

Kora Launcher does **not collect, store, or transmit personal data to any external servers**.

All data generated by the app is stored locally on your device and is used only to provide core functionality.

### Permissions Used:

* **Usage Access (UsageStats):**
  Required to detect which applications are opened and measure usage time. This enables features such as app usage tracking and intervention prompts.

* **Default Launcher Permission:**
  Required to replace the system launcher and provide the Kora Launcher interface.

* **Local Storage (SQLite Database):**
  Used to store:

  * App usage data
  * User-defined intentions
  * Interaction logs (e.g., app open attempts)

This data remains strictly on your device and is never transmitted.

## 2. Data Sharing

Kora Launcher does **not share, sell, or transmit any user data**.

* No data is sent to external servers
* No third-party analytics or tracking tools are used
* No advertising networks are integrated

## 3. Data Security

All data is stored locally on the user's device using standard Android storage mechanisms. No remote access to this data exists.

## 4. Future Features

If future versions of Kora Launcher include optional online features (such as AI-based suggestions), users will be explicitly informed and required to provide consent before any data is transmitted.

## 5. User Control

Users have full control over their data.

* Uninstalling the app will remove all stored data
* Clearing app data from device settings will permanently delete all stored information
